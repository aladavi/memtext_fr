Comment utiliser MemText

📝 Créer des notes

  1. Entrez un titre pour votre ensemble de mémorisation
  2. Collez votre texte avec une phrase par ligne
  3. Appuyez sur « Générer les notes »

📖 Étudier

  1. Commencez à l'étape 0 (texte complet)
  2. Lisez et mémorisez le contenu
  3. Passez à l'étape suivante (les mots disparaissent progressivement)
  4. Essayez de vous rappeler les mots manquants
  5. Continuez jusqu'à pouvoir tout réciter à partir des tirets

📋 Utiliser avec Apple Notes

  1. Créez un dossier « mem - [Votre titre] » dans Notes
  2. Pour chaque étape, appuyez sur « Copier pour l'app Notes »
  3. Créez une nouvelle note, collez le contenu
  4. Utilisez le titre de note affiché dans l'app
  5. Après avoir créé toutes les notes, ajoutez les liens entre elles
